# MCP-ready Workflows

API20 explores how x402-powered APIs can be discovered, documented, and prepared for future MCP-ready agent workflows.

The current version does not operate an official MCP server. MCP support is research/planned.

## What MCP means for API20

**MCP** (Model Context Protocol) defines how AI agents connect to tools and data sources.

API20 is designed to help developers discover and document APIs that may later be exposed to AI agents through MCP servers.

## Focus areas

* Agent tool discovery
* MCP server research
* Payment-aware API metadata
* API discovery for agent tools
* Future MCP server compatibility research

## Future flow

1. Developer lists an x402-powered API
2. API20 documents payment requirements
3. API metadata is prepared for agent/tool usage
4. Future MCP server compatibility exposes APIs to AI agents
5. Agents inspect paid resources before calling them
